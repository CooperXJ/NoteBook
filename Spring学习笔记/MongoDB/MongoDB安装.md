2. 打开bin目录并在此打开terminal

3. 指定创建数据库的位置

   `sudo ./mongod --dbpath=/Users/cooper/Documents/Code/Plugins/mongodb/data/db/`

   也可指定log的位置

4. 打开的端口号为27017

   出现该字样代表服务已经打开

   进入到bin目录下输入./mongo
